# Paisabai Tour Booking Web Application

A university group project focused on UI/UX design and frontend web development for a tour booking platform.

This project was developed by a team of 4 members during the first year of Computer Engineering studies.

## My Contributions

- Designed user interface (UI) and user experience (UX) using Figma
- Developed frontend UI pages
- Implemented:
  - Booking history page
  - Payment status page
  - Login and registration interface
- Participated in user flow and interface planning

## Features

- Tour package browsing
- Tour booking interface
- Payment status tracking
- Booking history system
- User authentication UI
- Responsive web interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Figma

## Project Structure

- `diagrams/` → Use case diagram and Figma flow
- `screenshots/` → UI screenshots and wireframes
